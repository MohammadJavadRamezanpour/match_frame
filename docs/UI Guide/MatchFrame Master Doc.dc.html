01 · Philosophy

Darkroom & Spotlight

Photographers keep their workspace a neutral grey so colour judgement stays honest. So do we: the environment is calm and neutral, and one warm gold tone is reserved for a single job, marking the best photo. Attention goes exactly where it matters.

{{ p.body }}

Not a red/pink dating app. Those sell excitement; we build trust. Our personality is a calm, honest, kind professional advisor.

03 · Color tokens

Three roles, two modes

Every colour is a CSS variable, never hardcoded in a component. Neutral carries the interface, Iris is the brand, and gold spotlight is reserved only for the best photo and its badge.

04 · Typography

Two modern sans, a precise mono

Outfit gives the friendly, confident voice in big headings. IBM Plex Sans carries the whole UI. Geist Mono is for numbers, votes, percentages, ranks, so data reads like a precise instrument.

06 · Signature element

The viewfinder focus brackets

Four corner brackets, like a camera's focus indicator, plus a soft gold halo wrap the #1 photo, and nothing else. It echoes the photography concept and is the one bold gesture in an otherwise calm interface. Never repeat it elsewhere.
